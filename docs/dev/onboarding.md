# Developer Onboarding

1. Clona repo e configura `.env`.
2. Esegui `make setup`.
3. Esegui `make bootstrap-auth-dev`.
4. Avvia stack locale con `make demo`.
5. Verifica gate minimi: `make lint`, `make test`, `make audit`.
