## Note on links and numbering

Internal relative links inside these files point at their **original**
`docs/plan/` location and may not resolve from `archive/`. They are not
maintained. Any "Phase 8/9/10" inside `three-month-code-plan.md` or
`swarmos-roadmap-phase7-30-draft.md` refers to the **retired** swarmos
sub-numbering, not the canonical evidence-to-scale phases.
